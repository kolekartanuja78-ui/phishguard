About This Project

Intelligent Phishing & QR Protection

SecureScan is a Phishing Website Detection System developed to protect users from malicious websites and fake QR links. It is designed as a cybersecurity project that uses machine learning techniques to classify URLs as safe or malicious.

🎯 Project Objective

Detect phishing websites in real-time and prevent users from accessing harmful content while browsing the internet.

🧠 Technology Used

✔ HTML, CSS & JavaScript
✔ Python (Flask / Streamlit)
✔ Machine Learning (MLP)
✔ URL Feature Extraction

🔬 Machine Learning Model

Multi-Perceptron model trained on phishing datasets with 94.2% accuracy.

🚀 Why This Project is Important

Protects users and organizations from fake login pages and malicious cyber threats.
